### 🚀 Overview
A high-performance automation suite built to ensure the reliability of the **OmniStore** platform. This project demonstrates a complete CI/CD lifecycle, moving from local Python scripts to cloud-based execution with interactive stakeholder reporting.

### 🛠️ Tech Stack
* **Language:** Python 3.11
* **Framework:** Playwright (Pytest-Playwright)
* **CI/CD:** GitHub Actions (Parallel Matrix Execution)
* **Reporting:** Allure Reports (Hosted via GitHub Pages)

### ✨ Key Features
* **Cross-Browser Testing:** Parallel execution on Chromium and Webkit (Safari).
* **Automated Reporting:** Live dashboard updates automatically on every code push.
* **Hybrid Testing:** Capable of handling both UI and API validation within the same suite.
* **Session Management:** (Coming Soon) Optimized login state sharing to reduce execution time.
